1.2.1 整型

    整型一般分为短整型（short）、长整型（long long），其中短整型（short）一般用不到。
　　①对于整型int来说，一个整数占用32bit（即32位），也即4Byte（即4字节），取值范围是)，一般来说，当绝对值在以内的整数都可以定义成int型。

    ②对于长整型long long来说，一个整数占用64bit（即64位），也即8Byte（即8字节），取值范围是)，一般来说，当绝对值超过2147483647（例如)或者）的整数都可以定义成long long型。

  注意，如果long long型赋值大于时，需要在数后加”LL”后缀。

    此外，无符号整型（unsigned）占用字节数与原类型一致，只是将取值范围扩大到之前正数部分的2倍。unsigned int取值范围是，unsigned long long取值范围是。
　　简单来说，如果数字在)以内或者32位整数，就使用int型来存放；如果数字在以内或者64位整数，就使用long long来存放。

  %d是int型的输出格式，%f是float和double型的输出格式。

1.2.2 浮点型

    浮点数就是小数，其中浮点型分为单精度（float）和双精度（double）。
　　①对于单精度float来说，一个浮点数占用32bit（即32比特），其中1bit为符号位，8bit为指数位，23位为尾数位，可以存放的数范围是，但精度只有6~7位。

    ②对于双精度double来说，一个浮点数占用64bit（即64比特），其中1bit为符号位，11bit为指数位，52位为尾数位，可以存放的数范围是，精度为15~16位。

  尽量不要用float，在需要处理浮点数时尽量使用double储存。

1.2.3 字符型

（1）字符变量和字符常量

    先看一个例子：

    如果通过char c;定义了一个字符，那么c在这里就被称作“字符变量”，它可以被赋值；如果是一个字符本身（例如小写字母’e’），他是一个没有办法被改变其值的东西，则将其称为“字符常量”。在C语言中，字符常量使用ASCII统一编码，其中小写字母比大写字母的ASCII码大32。
　　在C语言中，字符常量（必须是单个字符）必须用单引号标注。
    char c = 117;中字符变量’c’所指代的是字母’u’，因为u的ASCII码为117.

  %c是char型的输出格式。

（2）转义字符

    几个常用的转义字符”\n”表示换行；”\t”表示制表位，等同于Tab键位；”\0”表示空字符NULL，其ASCII码值为0。

（3）字符串常量

    字符串常量是由若干字符组成的串，C语言没有储存字符串的基本数据类型（C++中有string类型），可以使用字符数组进行存储。
    char str[] ="ABCD";用双引号进行标记，并使用%s格式进行输出。

  不能将字符串常量赋值给字符标量。

1.2.4 布尔型

    布尔型在C++中可以直接使用，但在C语言中必须添加stdbool.h才能使用。bool型变量只能是true或者false，赋值时可直接使用true和false，或者使用整数进行赋值。整数常量在赋值给布尔型变量时会自动转换为true（非零）和false（零）。对于计算机而言，true和false分别表示为1和0，故用%d输出bool型变量时会输出1和0。

1.4 符号常量和const常量

    符号常量通常来讲就是”替换”，即用一个标识符来替代常量，又称”宏定义”或者”宏替换”，格式如下：
    #define PI = 3.14
    末尾不加分号。
    另一种替换的方法是使用const，其格式如下：
    const double pi = 3.14;
    以上两种办法都被称为常量，但推荐const的写法。
    define可以定义任意片段：
    #define 标识符 任何语句或片段
    #define ADD(a,b) ((a)+(b))
    上述代码表示，定义一个ADD(a,b)来代替a+b。
    通过上述例子不难发现宏定义是直接将对应的部分替换，然后才进行编译和运行。替换时只是单纯的替换不包含优先级。如#define CAL(x) (x * 2 + 1)中，若令x = a + 1，则结果为a + 3，因为计算时x会直接替换为a + 1，则原来的式子变为a + 1 * 2 + 1，但是可以通过括号来确定优先级。
